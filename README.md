### os模块的函数
* os.remove()	删除文件
* os.makedirs()	创建多层目录
* os.rmdir()	删除单级目录
* os.rename()	重命名文件
* os.pathisfile()
* os.path.isdir()
* os.path.join()
* os.path.splitext()	将文件分割成文件名与扩展名，如分割tmp.txt为tmp和.txt
